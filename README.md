# gosns

## Overview
gosns is messaging server like Amazon SNS.  
Using messaging model is Pub-Sub messaging model.

## Install
```sh
% go get -u "github.com/midorigreen/gosns"
```

## Build
```sh
# build
% go build

# run
% ./gosns
```
